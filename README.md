# Finding Aid RAG Project
## ISTA 556 Project: Using retrieval augmented generation (RAG) to improve accessibility of digital archival finding aids for general audiences

# To set up API
Will require a [Claude API key](<https://platform.claude.com/docs/en/api/admin/api_keys/retrieve>). Set the API key in the .env file (see env example for template)

# To run the app
py -m streamlit run app.py

# About the app
Questions are treated as individual queries and history does not build off of previous questions
or responses.
